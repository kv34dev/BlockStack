# BlockStack 

This is a classic version of **Tetris**, written in Python using the Pygame library. The game features the basic gameplay of Tetris, including various block types, piece rotation mechanics, line clearing, and score tracking.

## Features

- **Random block generation** with different colors
- **Piece rotation** to fit pieces into the grid
- **Line clearing** and score accumulation
- **Game over** occurs when pieces don't fit into the game field

## Requirements

- Python 3.x
- Pygame (install with `pip install pygame`)

## Controls

- **Left Arrow** — move the piece left
- **Right Arrow** — move the piece right
- **Down Arrow** — speed up the piece's fall
- **Up Arrow** — rotate the piece

## Development

This project was created to learn the basics of working with the Pygame library. You are free to clone the repository and modify the game as you wish, such as adding new features, changing graphics, or improving the game logic.
